# 🚀 Desafio QA - Beedoo 2026

# 📌 Sobre o Projeto
Este repositório contém a análise técnica e o plano de testes para o módulo de cadastro e listagem de cursos da aplicação Beedoo.

# 🎯 Qual você acredita ser o objetivo da aplicação?
O sistema funciona como uma plataforma educacional para gerenciamento de cursos. Suas principais funções permitem o controle de:

Modalidade: Presencial ou Online.

Corpo Docente: Gestão de instrutores.

Cronograma: Definição precisa de datas de início e término.

# 🛣️ Quais são os principais fluxos disponíveis?
Criação de Cursos: Validação de campos obrigatórios e persistência de dados.

Listagem: Visualização e recuperação assertiva das informações cadastradas.

# ⚠️ Pontos Críticos (Foco do QA)
Para garantir a qualidade, priorizei os seguintes cenários:

Integridade da Listagem: Verificação se os valores exibidos na tabela correspondem exatamente ao que foi inserido no cadastro.

Integridade no cadastro: Verificação se as informações adicionadas no formulário vem com algum erro no endpoint ou tela.
